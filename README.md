# Decision Tree Regression App

A simple Machine Learning web application built using Streamlit and Scikit-learn for predicting diabetes progression using a Decision Tree Regression model.

## Live Demo

[Decision Tree Regression App](https://decisionstreeregression.streamlit.app/?utm_source=chatgpt.com)

---

## Features

- Decision Tree Regression Model
- Interactive Streamlit UI
- Real-time prediction
- User-friendly input fields
- Pickle model integration
- Simple deployment using Streamlit Cloud

---

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- NumPy
- Pickle

---

## Dataset Used

The model uses the Diabetes Dataset from Scikit-learn.

Features include:

- Age
- Sex
- Body Mass Index (BMI)
- Average Blood Pressure
- Total Serum Cholesterol
- Low-Density Lipoproteins (LDL)
- High-Density Lipoproteins (HDL)
- Total Cholesterol / HDL Ratio
- Serum Triglycerides Level
- Blood Sugar Level
